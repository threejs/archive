[method:String resolveURL]( [param:String url], [param:String path] )

[page:String url] — 绝对或相对 URL 解析。
[page:String path] — 要解析的相对 URL 的基本路径。

根据给定路径解析相对 URL。绝对路径、数据 URL 和 Blob URL 将按原样返回。无效 URL 将返回空字符串。
