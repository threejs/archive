KTX2导出器（[name]）

一个用于 `KTX2` 的导出器。

[link:https://www.khronos.org/ktx/ KTX2] 是一种开放格式规范容器格式([link:https://github.khronos.org/KTX-Specification/ktxspec.v2.html open format specification])，可高效存储和传输纹理数据，并支持 GPU 优化格式。

导入

[name] 是一个附加组件，必须显式导入。请参阅 [link:#manual/introduction/Installation Installation / Addons]。

import { KTX2Exporter } from 'three/addons/exporters/KTX2Exporter.js';

代码示例

// Instantiate an exporter const exporter = new KTX2Exporter(); // Export a data texture const result = await exporter.parse( dataTexture ); // Export a render target const result = await exporter.parse( renderer, renderTarget );

创建一个 [name] 实例。

方法

[method:Promise parse]( [param:DataTexture texture] )

[page:Object texture] — 要导出的 DataTexture 或 Data3DTexture。

从输入纹理生成 KTX2 文件。返回一个包含结果的 promise。

[method:Promise parse]( [param:WebGLRenderer renderer], [param:WebGLRenderTarget renderTarget] )

[page:Object renderer] — 用于从渲染目标读取像素数据的渲染器。需要是一个 [page:WebGLRenderer] 或 [page:WebGPURenderer]。
[page:Object renderTarget] — 要导出的渲染目标。需要是一个 [page:RenderTarget]。

从渲染目标生成 KTX2 文件。返回一个解析结果的 Promise。
